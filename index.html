Bot Guides

Step-by-step instructions to get your bot running fast.

1
Fork the Repository
Head to the GitHub profile and fork the bot repo you want. This creates your own copy you can edit freely.
→ github.com/popkidmain
2
Get Your Session ID
Visit the session generator, scan the QR with WhatsApp (Linked Devices), and copy the session string.
→ Open Session Generator
3
Configure the Bot
Edit your .env file with your session ID and preferences.
SESSION_ID=popkid_xxxxxxxxxxxxxx
BOT_NAME=ULTRA-MD
PREFIX=.
OWNER_NUMBER=254111385747
4
Install & Run
Install dependencies and start your bot locally to test before deploying.
npm install
npm start
1
Sign Up on Render
Create a free account at render.com and connect your GitHub account.
2
Create a Web Service
On the dashboard click New → Web Service and select your forked bot repository.
3
Add Environment Variables
In the Environment tab, add your config variables.
SESSION_ID → your_session_here
OWNER_NUMBER → 254111385747
BOT_NAME → ULTRA-MD
PREFIX → .
4
Set Commands
Build Command: npm install
Start Command: npm start
5
Deploy & Go Live 🎉
Click Create Web Service. Render builds and launches your bot 24/7 for free!
1
Visit the Generator
Open the session ID link in your browser.
→ popkidxmdscanninglink.onrender.com
2
Scan the QR Code
Open WhatsApp → 3 dots (⋮) → Linked Devices → Link a Device → Scan.
3
Copy Your Session ID
A string starting with popkid_ appears. Copy the entire thing.
popkid_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
4
Keep It Private ⚠️
Your session ID = access to your WhatsApp. Never share it publicly.
⚠️ Store as an environment variable ONLY. Never hardcode it in your source files.
